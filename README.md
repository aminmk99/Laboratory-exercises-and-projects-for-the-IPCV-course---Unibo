## 📂 Repository Structure

### 🧪 Lab Sessions

1. **Spatial Filtering & Edge Detection** 🔍
   - **Concepts:** Understanding images as NumPy tensors, coordinate systems, and color space conversions (BGR ↔ RGB).
   - **Filtering:** Implementation of linear and non-linear filters including **Mean**, **Median**, and **Gaussian** blurs for noise reduction.
   - **Advanced Denoising:** Exploring the **Bilateral Filter** for edge-preserving smoothing.
   - **Edges:** Using the **Canny Algorithm** and Hysteresis thresholding to extract structural outlines.
